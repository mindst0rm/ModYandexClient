# Что нового

## ✨ Проект продолжает жить
- Разработка `ModYandexClient` теперь официально продолжается в репозитории [mindst0rm/ModYandexClient](https://github.com/mindst0rm/ModYandexClient).
- Клиент остаётся самостоятельным: регулярные обновления будут выходить и для тех, кто не хочет ставить дополнительный софт.

## 🚀 Launcher как опция, а не обязанность
- Если нужен максимально быстрый автоапдейт при запуске, можно использовать [yamusic-launcher](https://github.com/mindst0rm/yamusic-launcher).
- Если Launcher не нужен, обычный клиент можно продолжать использовать как и раньше.

## 🛠 Что изменилось в этой версии
- Убраны упоминания PulseSync и сценарии миграции на сторонний проект.
- Стартовое окно теперь рассказывает о новом репозитории и отдельно предлагает Launcher как удобный, но необязательный способ обновления.
- Ссылки на репозиторий обновлены в `toolset.js`, runtime-апдейтере, FFmpeg updater'е, документации и пользовательских сообщениях внутри клиента.

## 📦 Для пользователей
- Обновления для клиента будут продолжаться регулярно.
- Дополнительные утилиты больше не навязываются.
- Launcher нужен только в том случае, если вы хотите полностью автоматизировать обновление при запуске.

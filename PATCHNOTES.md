# Что нового

## 🛠 Улучшена работа с FFmpeg
- Добавлена явная проверка установленного FFmpeg при запуске приложения.
- После установки FFmpeg теперь дополнительно проверяется контрольная сумма и сам запуск бинарника.

## 🔔 Что теперь увидит пользователь
- Если FFmpeg не установлен, приложение попытается установить его автоматически.
- Если установка или проверка FFmpeg завершится ошибкой, приложение покажет понятное уведомление.
- Если FFmpeg окажется недоступен уже во время скачивания трека, ошибка больше не будет скрываться молча.

## 📦 Для пользователей
- Базовая версия клиента обновлена до `5.94.0`.
- Launcher по-прежнему можно использовать для удобного автообновления: [yamusic-launcher](https://github.com/mindst0rm/yamusic-launcher).
